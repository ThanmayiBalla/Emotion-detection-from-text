# AI-Driven Emotion Detection & Personalized Learning Support Platform

An intelligent, production-ready, multi-model hybrid student tracking interface. It captures students' written queries, predicts emotional states (Bored, Confident, Confused, Curious, Frustrated) using customized Deep Learning models (BiLSTM & DistilBERT), matches keyword corrections, and queries Gemini LLM for personalized study schedules.
